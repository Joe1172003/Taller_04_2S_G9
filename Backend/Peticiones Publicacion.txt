Peticiones de Publicaciones 

Agregar una publicacion : POST -> http://localhost:3800/publication/addPublication
Ejemplo     # Nota puede venir el campo "course" o el campo "professor" uno de los dos, o los dos pero no puede venir los dos vacios
{
    "course": "Fisica 2"
    "professor": "Cuyan", 
    "message": "Realiza ejemplos practicos 3"
}


Eliminar una publicacion : DELETE -> http://localhost:3800/publication/deletePublication/id
                             ejemplo http://localhost:3800/publication/deletePublication/2

		Nota: Solo la podra eliminar la publicación quien creo dicha publicación 

Modificar una publicacion : PUT  -> http://localhost:3800/publication/updatePublication/id
                            ejemplo http://localhost:3800/publication/updatePublication/7

		Nota: Solo la podra eliminar la publicación quien creo dicha publicación 

Listar Publicaciones : GET -> http://localhost:3800/publication/getPublications

# Filtro de Busqueda 

Listar Publicación SOLO las que tengan Curso: GET -> http://localhost:3800/publication/getPublicationsFilterByCurso
Listar Publicación SOLO las que tengan Catedratico: GET -> http://localhost:3800/publication/getPublicationsFilterByCatedratico

Listar Publicación Por Nombre del Curso GET -> http://localhost:3800/publication/getPublicationsFilterByNameCurso
Ejemplo
{
    "nameCurso": "Apli 3"
}

Listar Publicación Pir Nombre del Catedratico GET -> http://localhost:3800/publication/getPublicationsFilterByNameCatedratico
Ejemplo
{
    "nameCatedratico": "Erwin Peñate"
}
