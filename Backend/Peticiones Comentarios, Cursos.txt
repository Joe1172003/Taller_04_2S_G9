###### Comentarios ########


Agregar Comentario: POST-> http://localhost:3800/comment/addComment/<id PUBLICACION>
		    Ejemplo: http://localhost:3800/comment/addComment/2
Ejemplo body:
{
    "message": "Comentario prueba id 2 -2"
}

Obtener comentarios de una publicacion: GET-> http://localhost:3800/comment/getComments/<id PUBLICACION>
			        Ejemplo:http://localhost:3800/comment/getComments/1

Eliminar Comentario: DELETE-> http://localhost:3800/comment/deleteComment/<id COMENTARIO>
		     Ejemplo: http://localhost:3800/comment/deleteComment/3

Modificar Comentario: PUT-> http://localhost:3800/comment/updateComment/<id COMENTARIO>
		   Ejemplo: http://localhost:3800/comment/updateComment/6
Ejemplo body:
{
    "message": "COMENTARIO MODIFICADO"
}


####### CURSOS ##########
Agregar cursos Aprobados: POST-> http://localhost:3800/course/addCourse/<REGISTRO ACADEMICO>
			         http://localhost:3800/course/addCourse/202200271

Obtener cursos de un Usuario: GET-> http://localhost:3800/course/getCourse/<REGISTRO ACADEMICO>
                                    http://localhost:3800/course/getCourse/202200271

Eliminar curso de un Usuario: DELETE-> http://localhost:3800/course/deleteCourse/<id CURSO>
				       http://localhost:3800/course/deleteCourse/1
